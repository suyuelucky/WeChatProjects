# 绣花针项目 - 拍照写留痕和编辑页

## 项目概述
本项目是绣花针平台的核心功能模块，旨在提供直观、高效的工作记录方式，实现"拍照即留痕，说话即记录"的便捷体验。通过优化信息采集、编辑和发布流程，显著提升基层工作效率。

## 技术栈
- 微信小程序
- 腾讯云托管
- Node.js v18.15.0
